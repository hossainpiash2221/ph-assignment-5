What is the document object model(DOm)?

The Document Object Model (DOM) is a programming interface that represents the structure of a web document as a tree of objects. It allows developers to access, modify, and manipulate HTML and XML documents dynamically using languages like JavaScript.

What is event delegation in the context of the dom , and why is it useful?

Event delegation refers to the process of using event propagation (bubbling) to handle events at a higher level in the DOM than the element on which the event originated. It allows us to attach a single event listener for elements that exist now or in the future.
